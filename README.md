# 초민정음 MK2 — AI 한글 & 초등 국어 교육 플랫폼

> [초민정음(jammin)](../jammin)을 현대적으로 재설계한 차세대 한글 및 초등 국어 교육 플랫폼. 웹과 모바일을 모두 지원하며, AI를 활용한 맞춤형 학습 경험을 제공합니다.

## 배경

기존 `jammin`(초민정음)은 Spring Boot + Thymeleaf + jQuery 기반의 초등 한글 학습 웹앱으로, 받아쓰기 학습지 생성과 한글 분해 기능을 제공했습니다. 그러나 다음과 같은 한계가 있었습니다:

- 데스크톱 브라우저에 최적화된 UI로 모바일 사용이 불편
- 정적 콘텐츠 중심으로 개인 맞춤형 학습 불가
- 서버 사이드 렌더링(SSR)으로 인터랙티브한 학습 경험 제한

## 핵심 기능 (계획)

### AI 기반 학습
- **맞춤형 받아쓰기** — AI가 학생의 수준을 분석하여 적절한 난이도의 받아쓰기 문제 자동 생성
- **발음 교정** — 음성 인식 + AI 분석으로 발음 교정 피드백
- **글쓰기 도우미** — AI가 학생의 글을 분석하고 문법, 맞춤법, 표현력 피드백
- **학습 리포트** — 학습 이력을 기반으로 강점/약점 분석 리포트 생성

### 한글 학습
- **자모 학습** — 자음, 모음 학습 및 조합 연습
- **획순 연습** — SVG 애니메이션 기반 획순 가이드
- **받아쓰기** — 수준별 받아쓰기 학습지 생성 및 자동 채점
- **긴글 연습** — 단계별 문장 쓰기 연습

### 초등 국어
- **교과서 연계** — 초등 국어 교과서 단원별 학습 콘텐츠
- **독해력 향상** — AI 기반 독해 문제 생성 및 해설
- **어휘 확장** — 맥락 기반 새 어휘 학습
- **교사 도구** — 학습지/교안 자동 생성, 학급 관리

### 크로스 플랫폼
- **반응형 웹** — 데스크톱, 태블릿, 모바일 최적화
- **모바일 앱** — Flutter 기반 iOS/Android 네이티브 앱
- **오프라인 지원** — 핵심 학습 콘텐츠는 오프라인에서도 사용 가능

## 기술 스택 (계획)

| 구분 | 기술 |
|------|------|
| 백엔드 | Java 21, Spring Boot 3.x, Spring AI |
| 프론트엔드 | Vue 3, TypeScript, Vite, Tailwind CSS |
| 모바일 | Flutter (iOS/Android) |
| AI | Google Gemini / Claude (학습 분석, 콘텐츠 생성) |
| 음성 | Web Speech API / Google STT |
| DB | PostgreSQL, Redis |
| 빌드 | Gradle (백엔드), npm (웹), Flutter (모바일) |

## 기존 자산 활용

`jammin`에서 가져오는 것들:
- **한글 분해 유틸** (`HangulUtil.java`) — 음절 → 초성/중성/종성 분해 로직
- **한글 SVG 자산** — 글자별 SVG 파일 (획순 연습용)
- **교육용 폰트** — KCC 도담도담체 등

## 로드맵

| 단계 | 내용 |
|------|------|
| Phase 1 | 웹 플랫폼 기초 — 반응형 UI, 기존 jammin 기능 마이그레이션 |
| Phase 2 | AI 통합 — 맞춤형 문제 생성, 자동 채점, 학습 분석 |
| Phase 3 | 모바일 앱 — Flutter 앱, 음성 인식, 오프라인 지원 |
| Phase 4 | 교사 대시보드 — 학급 관리, 학습 리포트, 교안 생성 |
